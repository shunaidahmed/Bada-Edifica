# Bada Edifica - Professional Building Solutions

A modern, responsive website for Bada Edifica, a construction and building company based in Barcelona, Spain. Built with pure HTML, CSS, and JavaScript using the Renault design system as inspiration.

## About

Bada Edifica is a construction and building company founded in 2023 in Barcelona, Spain. The company handles every part of a building project under one roof:
- Electrical Installations
- Plumbing Installations
- Construction & Renovations
- Site Cleanliness
- Fire Safety Systems

## Features

- **Modern Design System**: Based on Renault's design language with #912526 as primary accent color
- **Fully Responsive**: Works on desktop, tablet, and mobile devices
- **Hero Slider**: Interactive slider with 4 slides, auto-advance, and navigation controls
- **Mega Menu**: Advanced dropdown menu for services navigation
- **Smooth Animations**: Scroll-triggered fade-up animations and counter animations
- **Oswald Font**: Consistent typography throughout the site
- **Background Patterns**: Subtle dot, grid, and diagonal patterns on sections

## Pages

- **index.html** - Homepage with hero slider, about preview, services, timeline, values, and more
- **about.html** - About Us, Our Story, Journey Timeline, Values, Mission & Vision
- **services.html** - Detailed services page with 6 service areas
- **projects.html** - Portfolio of completed projects
- **contact.html** - Contact form and company information

## Technologies Used

- **HTML5** - Semantic markup

- **JavaScript (ES6+)** - Vanilla JS for interactivity
- **Google Fonts** - Oswald font family
- **Design System** - Renault-inspired design tokens

## Design Tokens

### Colors
- Primary: `#912526` (Bada Edifica Red)
- Canvas: `#ffffff` (White)
- Surface Dark: `#000000` (Black)
- Surface Soft: `#f7f7f7` (Light Grey)

### Typography
- Font Family: Oswald (all text)
- Display XL: 56px / 700 weight
- Body: 15px / 400 weight

### Spacing
- Section Padding: 100px
- Container Max Width: 1440px
- Border Radius: 2px (buttons), 0px (cards)

## Setup

1. Clone or download the repository
2. Open `index.html` in a web browser
3. No build process required - pure HTML/CSS/JS
